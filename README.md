# Freelance Job Marketplace — Week 1 (Full Pages) — React JSX + Tailwind + MongoDB Cloud

This package includes a **full set of wireframe pages** (placeholders) for all roles so you can iterate week-by-week.
It uses **React (JSX)**, **Tailwind CSS (only)**, and **MongoDB Atlas**.

## Quick Start (npm)
```bash
# Install
npm --prefix client install
npm --prefix server install

# Dev
npm --prefix client run dev      # http://localhost:5173
npm --prefix server run dev      # API http://localhost:8080

# Configure server (copy .env.sample → .env and set MONGODB_URI)
```
